Instructions for Using the Tool to Perform a Denial of Service Attack

    Note: Ensure that the name of the external network card used is "wlan1" before starting.

Step 1:
Run the tool by executing the command "bash deauth" and then enter your computer's username.


Step 2:
The tool will then display a list of available networks. Wait until the desired network appears, then press "Ctrl + C" to stop the display.


Step 3:
Enter the number of the network you want to attack.


Step 4:
A list of devices connected to the chosen network will appear. When the desired device appears, press "Ctrl + C" to stop the display.


Step 5:
Enter the number of packets you want to use for the attack. If you want to perform the attack indefinitely, enter the number zero.
